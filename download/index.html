Changelog

Usagi 0.0.2

This is Usagi, second stable version - 0.0.2. It's not just a fork. It's even better than you think.

What's different

We focused on providing bug fixes for the core features of version 0.0.1 and fixed several crashing issues in the original application, while also adding new features to make reading and accessing content easier.

Good news: Usagi has officially added support for older Android devices running Android 5.0 and 5.1. We've also worked to refine the source code and minimize resources to ensure Usagi remains as lightweight and smooth as possible. Known bugs such as application crashes or inability to access content, including "Bad position" have also been completely resolved.

Happy reading!

What's changed

  • Added bypass firewall / internet censorship feature
  • Added some plugin migrate methods to avoid app crashes
  • Added plugins management in settings, allow external plugins
  • Brought back compatibility with Android 5.0+ devices
  • New display mode added: Cover only (Hide title in grid list)
  • Enabled tap to view and save backdrop images
  • Restored the welcome screen, removed intergrated content
  • Updated app icon (background color), dependencies, translations and added new values / strings
  • Fixed several memory leak issues and some backward compatibility issues
  • Resolved app crashes in certain cases, especially on Android 5 + 6

Thank you everyone for following and using Usagi. Thank you for your support and contributions.
